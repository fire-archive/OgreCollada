OgreCollada Components
----------------------

* OgreCollada library
* OgreCollada tools (Viewer and Converter)
* OpenCOLLADA components (OgreCollada dependency)

When the components have been successfully built, they can be found in the
following directories:
OgreCollada library: lib
OgreCollada tools: tools/bin
OpenCOLLADA library components: lib

OgreCollada Dependendencies
---------------------------
The direct dependencies listed here are the versions against which OgreCollada was
tested.
 * Boost v1.35.0-r2 (www.boost.org): boost_system, boost_regex, boost_filesystem
 * Ogre3D Engine v1.6.1 (www.ogre3d.org)
 * libxml2 v2.7.3-r1 (www.xmlsoft.org)
